Protect the Castle CTF

Welcome to the homepage for Protect the Castle CTF, hosted by HHS-Cyber.

Purpose

PTCCTF was created for educational purposes by the Huntsville High School Cybersecurity department, HHS-Cyber. Through competition, HHS-Cyber hopes to introduce competitors to new topics, allow for tool experimentation, and increase interest in cybersecurity.

Problem Types

Networking
Problems centered around using networking commands and tools
Web
Problems focused on navigating and penetrating websites
Forensics
Problems involving deciphering large amounts of data
Crypto
Problems about uncovering hidden information from files
Code
Problems related to understanding and changing programming and code
Misc.
Problems that do not fall in a specific category
